## 什么是 Watchtower？

Watchtower 是一个开源的自动化解决方案，用于监控 Docker 容器的基础镜像更新，并在检测到新版本时自动更新容器。

## 主要特性

- **自动监控**：持续监控 Docker Hub 和私有仓库的镜像更新
- **零停机更新**：智能重启容器，确保服务连续性
- **灵活配置**：支持多种配置选项和过滤规则
- **通知集成**：支持 Email、Slack、Microsoft Teams 等通知方式
- **安全连接**：支持 TLS 和认证机制
- **生命周期钩子**：支持更新前后的自定义脚本
